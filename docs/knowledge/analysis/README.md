# Deliverable di analisi Watchverse

Questa cartella contiene i deliverable prodotti dai subtask di Analisi e Progettazione Jira.

Ogni documento unisce analisi funzionale e analisi tecnica della Story e costituisce il riferimento per i subtask Front-End, Back-End, Cloud e QA collegati.

Il commento Jira del subtask di analisi deve collegare il documento corrispondente. Il documento viene aggiornato solo quando cambiano requisiti o decisioni; il collaudo manuale resta tracciato sulla Story principale.

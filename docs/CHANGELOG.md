# Watchverse Changelog

This project follows Semantic Versioning. The current product baseline is `1.0.0`; build numbers are generated independently for each published build and shown in the application footer.

## 1.0.0

- Cloud-backed account, profiles and library data through Supabase.
- Progressive metadata synchronization with manual retry for unresolved titles.
- Public metadata proxy support for TMDB, streaming availability, trailers and cinema data.
- Responsive Watchverse Black, Buffy and The Last of Us themes.
- Import/export support for TV Time GDPR archives, including rating conversion to the 1-10 Watchverse scale.
- Search, film and series libraries, profile PIN protection, accessibility documentation and design-system documentation.
- Loading-state orchestration for authentication, navigation and background library work.
